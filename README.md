# Ferrograph Studio 8 - Arduino Elapsed Time Indicator Replacement

### Overview
This repository contains the documentation and firmware for an Arduino Mega-based replacement for the Elapsed Time Indicator (timer/counter) section of the Ferrograph Studio 8 reel-to-reel tape recorder.  

This solution is designed to replace the custom unobtainium counter/memory IC ECTR4010 (IC425) and several ICs on the original logic board. It directly interfaces with the original board to handle tape transport pulse counting, tape direction monitoring, 7-segment display multiplexing, and BCD output generation.
